# Lab 02 - Pthreads

## Como Compilar:
Escrever o comando de compilação do C no terminal linux + nome do arquivo, que gerará um arquivo a.out:
gcc MultiplicacaoPthreads.c

## Como executar:
./nome do arquivo de saída:
./a.out
